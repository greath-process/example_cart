# Простейший компонент корзины
- Компонент отображает товары корзины + обновление количества и удаление товара из корзины
- Логика в class.php, использование D7
- Отображение в минимальном стиле. Построчное отображение товаров.

### По каждому товару вывод:
- изображение, из основной карточки товара - ссылка на товара;
- наименование - ссылка на товар;
- свойства товара
- поле ввода количества;
- цена/цена со скидкой;
- итого;
- ссылка для удаления товара из корзины.

## Структура:
- Весь js в script.js шаблона компонента с помощью jquery.
- Компонент работает в ajax режиме при манипуляции с составом корзины. Реализовано через script.js шаблона компонента.
- Стили - отдельный css файл. Минимальный, без изяществ.
- Скорость работы компонента, минимальное количество запросов.
- Не является копией стандартного компонента. Только мой код.
